# MovieSearch App

**Find Movies You'll Enjoy Without the Hassle**

A React-based movie search application that helps users discover films by filtering out unwanted genres like Drama and Romance. Built with modern web technologies for a fast and responsive experience.

## Features

- **Smart Search**: Instantly search for movies with optimized performance.
- **Genre Filtering**: Automatically excludes Drama and Romance genres for a tailored experience.
- **Trending Movies**: See popular movies based on community searches.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop.
- **Fast & Lightweight**: Quick loading times for an enjoyable user experience.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Appwrite for data storage and analytics
- **API**: The Movie Database (TMDB)
- **Icons**: Lucide React

## Quick Start

1. Clone the repository: `git clone https://github.com/your-username/movie-search-app.git`
2. Install dependencies: `npm install`
3. Set up environment variables (TMDB API key and Appwrite credentials)
4. Run the app: `npm run dev`
5. Open http://localhost:5173 in your browser
